# 미션 - 숫자 야구 게임

## 구현 기능 목록

1) 컴퓨터가 랜덤으로 숫자를 내는 기능
2) 입력받은 숫자에 대한 정합성 체크
    - 입력받은 값이 전부 숫자로 이루어져있는지
    - 입력받은 값이 세자리인지
3) 입력받은 숫자를 판단하는 기능
    - 볼의 개수를 판단하는 기능
    - 스트라이크의 개수를 판단하는 기능
    - 낫싱을 판단하는 기능
4) 입력받은 숫자에 대한 결과를 출력하는 기능
    - 스트라이크가 3개이면 모두 맞혔다는 문구 출력하는 기능
    - 그 외의경우 상기 3.의 결과값을 출력하는 기능 
5) 게임종료 후 재시작과 종료를 선택하는 기능
6) 게임을 최종 종료하는 기능
7) 게임을 재시작하는 기능
